Topic 01: AI-Powered Phishing Detection System  
Design a system that can detect and prevent modern phishing attacks across emails, URLs, 
and attachments in real time. The system should go beyond static rules to identify AI-generated 
phishing, multi-stage attack flows, and unknown threats. 
It must analyze both content and behavior to understand how attacks evolve. 
Participants should focus on building an adaptive system that can assign risk and explain its 
decisions clearly. The goal is to enable fast, accurate, and trustworthy threat detection.  
What to Build: 
● Combine content, behavioral, and contextual analysis 
● Build a risk scoring engine across multiple signals 
● Detect multi-stage attack chains (email → link → payload) 
● Incorporate attachment analysis (static/dynamic) 
● Add explainability for every detection decision 
● Enable real-time detection and response 
● A system that analyzes phishing across multiple vectors 
Optional Brownie Points: 
● Detect AI-generated phishing patterns (LLM fingerprints) 
● Campaign-level clustering of similar phishing attempts 
● Sandbox simulation for attachment behavior
